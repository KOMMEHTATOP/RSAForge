# RSAForge 🛡️

**RSAForge** — это легковесное Windows-приложение для генерации и валидации пар ключей RSA. Создано с акцентом на удобство разработчиков, которым нужно быстро получить ключи в специфических форматах для конфигов или тестов.

## ✨ Особенности
* **Разнообразие форматов:** Поддержка PKCS#8 (Modern), PKCS#1 (Legacy), XML и экспорт для SSH.
* **Гибкое форматирование:** Возможность мгновенно убрать заголовки (headers/footers) или схлопнуть ключ в одну строку (single-line) для удобной вставки в JSON/ENV.
* **Встроенная валидация:** Проверка пары ключей на математическое соответствие через тестовое шифрование.
* **Умное сохранение:** Автоматическое создание именованной папки с парой ключей (.key и .pub).
* **Современный интерфейс:** Написано на WPF с использованием библиотеки [WPF-UI](https://github.com/lepoco/wpfui) в стиле Windows 11 (Mica effect).

## 🛠 Технологии
* **Язык:** C# 10 / .NET 10
* **Паттерн:** MVVM
* **Библиотеки:**
    * `Wpf.Ui` — для Fluent дизайна.
    * `System.Security.Cryptography` — для работы с RSA.

## 🚀 Как запустить
Вы можете скачать готовую сборку в разделе **Releases**.
Приложение является **Self-contained** — это значит, что вам не нужно устанавливать .NET, всё необходимое уже находится внутри одного `.exe` файла.
